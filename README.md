# Daily Logs

An iPhone app for quickly recording wake-up time, bedtime, meals, and showers.

## Status

This project is in active development.

Current focus:
- ultra-simple daily logging flow
- clean, minimal iPhone-first UI
- Apple sign-in and guest mode
- lightweight analytics for sleep, meals, and showers

## Tech

- SwiftUI
- XcodeGen
- iPhone only
